### Installation
1. Clone this repository to local
```bash
git clone https://github.com/syifaull/bgn-react
```
2. Move to folder
```bash
cd bgn-react
```
3. Install dependencies
```bash
npm install
```
</br>

### Run your code
```bash
npm start
```
